# Tercer Avance
 
## Comandos de Compilación
 
En este apartado se explicara un poco de los comandos más básicos para
lograr que este programa logre compilar y ejecutar los códigos a disposición.
 
Como primer paso necesitamos crear un Makefile el cual será el responsable de contener
los comandos de compilación para nuestras máquinas, para esto es necesario ejecutar
el comando `untitled.pro`.
 
Una vez ejecutado el anterior comando se nos va a crear un archivo Makefile, el cual ejecutaremos
desde la terminal con el comando `make`, una vez ejecutados ya contaremos con el ejecutable.
 
Además como dato adicional podemos utilizar el comando `make clean` para poder volver a compilar el
programa.
 
## Comandos de Ejecución
 
Para este apartado se pretende mostrar al usuario la forma adecuada de cómo ejecutar el programa,
Para poder realizar este proceso se requiere antes haber ejecutado los comandos de compilación.
 
Una vez cumplidos los requisitos se debe ejecutar desde la terminal el comando `./untitled`
